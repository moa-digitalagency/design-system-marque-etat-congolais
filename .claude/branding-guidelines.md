# Official RDC Government Branding Guidelines

**Last Updated:** 2026-03-26
**Authority:** Marque État Congolais / Government of the Democratic Republic of Congo
**Design System:** design-system-marque-etat-congolais

---

## 1. LOGO ARCHITECTURE

### Three Indissociable Elements

The official logo consists of **THREE elements that are INSEPARABLE**. None can be used alone or modified independently:

#### 1.1 Bloc-Armoirie (Shield/Coat of Arms)
- **Role:** Static, immutable element representing state authority
- **Asset:** `armoiries.png` (color) | `armoiries_white.png` (white monochrome)
- **Aspect Ratio:** Square or near-square
- **Cannot be modified, resized disproportionately, or removed**

#### 1.2 Ligne d'État (State Bar)
- **Role:** Separator between shield and official title; symbolizes national unity
- **Asset:** `ligne_etat.png` (color) | `ligne_etat_white.png` (white monochrome)
- **Direction:** Primarily vertical (can be horizontal in decorative contexts)
- **Composition:** Tricolor bar — Bleu / Jaune / Rouge (33.33% each, top to bottom)
- **Width (Unit X):** 4px (digital) | 1mm (print minimum)
- **Fixed Height (Digital):** 53px (absolute minimum) — NOT proportional, NOT variable
- **Aspect:** Cannot be stretched, compressed, or deformed

#### 1.3 Intitulé Officiel (Official Title/Ministry Name)
- **Role:** Variable element identifying the government institution
- **Content:** Ministry, department, or institutional name in French
- **Font:** Principal font (sans-serif, typically the design system's primary font)
- **Style:** Uppercase, 6-8px font size (adjusted by context)
- **Line Breaks:** Maximum 5 lines (extensible intitulé)
- **Cannot be omitted or replaced with generic text**

## 2. COMPOSITION RULES

### 2.1 Security Zone (Zone de Sécurité)

- **Size:** 4X the width of the Ligne d'État
- **Calculation:** X = 4px → 4X = 16px
- **Rule:** A minimum of 16px clearance must surround the complete logo on all sides
- **Content:** No other design elements may encroach on this zone
- **Purpose:** Ensures visual integrity and prevents clutter

### 2.2 Unit System (Unité X)

- **Base Unit:** X = width of the Ligne d'État = 4px (digital)
- **Proportional Calculations:** All logo proportions derive from this unit
- **Security Zone:** 4X = 16px
- **Spacing Between Elements:** 4X = 16px
- **Purpose:** Maintains mathematical proportionality across all implementations

### 2.3 Height Equivalence

- **Rule:** Height of Ligne d'État = Height of Bloc-Armoirie
- **Alignment:** Elements align to the same baseline
- **Visual Balance:** Creates horizontal visual stability

### 2.4 Spacing Rules

- **Between Bloc-Armoirie and Ligne d'État:** 4X (16px)
- **Between Ligne d'État and Intitulé:** 4X (16px)
- **Around Complete Logo:** 4X (16px) minimum on all sides

---

## 3. SIZING GUIDELINES

### 3.1 Digital Sizes

| Context | Height | Asset Sizes | Badge |
|---------|--------|-------------|-------|
| **Web Header** | 50px | Armoiries: 50px<br>Ligne: 3px width, 50px height | Recommended |
| **Social Media / Video** | 110px | Armoiries: 40px<br>Ligne: 2px width, 40px height | Recommended |
| **Small Contexts** | 35px | Armoiries: 35px<br>Ligne: 2px width, 35px height | Standard |
| **Minimal** | 53px | Ligne d'État: 53px (height minimum) | Absolute Min |

**Rule:** Never scale below 53px total height. The Ligne d'État maintains minimum 53px in digital formats.

### 3.2 Print Sizes

- **Minimum Ligne d'État Width:** 1mm
- **Minimum Ligne d'État Height:** 14mm
- **Scaling:** Maintain proportions; never distort

### 3.3 Responsive Scaling

- **Mobile (< 768px):** Use 35-40px logo height
- **Tablet (768px - 1024px):** Use 50px logo height
- **Desktop (> 1024px):** Use 50-110px logo height (context-dependent)

## 4. AUTHORIZED VERSIONS

### 4.1 Version 1: Color on White (PRIMARY - RECOMMENDED)

- **Usage:** Default, preferred version for all contexts
- **Components:**
  - Armoiries: Colored (`armoiries.png`)
  - Ligne d'État: Colored (`ligne_etat.png`)
  - Intitulé: Black or dark text
- **Background:** White or very light backgrounds
- **Status:** ✅ **ALWAYS USE THIS VERSION FIRST**

### 4.2 Version 2: White Monochrome on Dark Background (AUTHORIZED)

- **Usage:** Dark backgrounds, colored headers, footers, banners
- **Components:**
  - Armoiries: White (`armoiries_white.png`)
  - Ligne d'État: White (`ligne_etat_white.png`)
  - Intitulé: White text
- **Background:** Dark or colored (navy blue, RDC blue, dark gray, etc.)
- **Status:** ✅ **AUTHORIZED FOR SPECIFIC CONTEXTS**

### 4.3 Version 3: Black Monochrome (RESTRICTED)

- **Usage:** Black and white printing only
- **Components:**
  - Armoiries: Black
  - Ligne d'État: Black (#323230 or pure black)
  - Intitulé: Black text
- **Background:** White only
- **Status:** ⚠️ **RESTRICTED USE — Print Only**
- **Note:** Not for digital; only for black/white documents

## 5. FORBIDDEN USAGES (STRICTLY PROHIBITED)

### 5.1 Proportional Deformations
- ❌ **NEVER** stretch, compress, or distort the logo
- ❌ **NEVER** change aspect ratios of elements
- ❌ **NEVER** resize elements independently
- **Consequence:** Violates brand integrity standards

### 5.2 Color Inversions
- ❌ **NEVER** invert the Ligne d'État colors (e.g., Rouge/Jaune/Bleu order)
- ❌ **NEVER** swap colors or use incorrect color order
- **Correct Order (Top to Bottom):** Bleu → Jaune → Rouge
- **Consequence:** Misrepresents national colors and violates protocol

### 5.3 Element Separation
- ❌ **NEVER** use elements separately (without the complete logo)
- ❌ **NEVER** use only the Bloc-Armoirie
- ❌ **NEVER** use only the Ligne d'État in logo contexts (decorative use is different)
- ❌ **NEVER** use only the Intitulé
- **Exception:** Ligne d'État and Armoiries can be used alone as **decorative elements** (not as the official logo)

### 5.4 Text Modifications
- ❌ **NEVER** change or abbreviate the institutional name
- ❌ **NEVER** use generic text like "Gouvernement" without specific ministry
- ❌ **NEVER** translate the official name (use French as specified)
- **Rule:** Use exact, current ministry or department names

### 5.5 Filter and Effect Restrictions
- ❌ **NEVER** apply shadows, glows, or special effects
- ❌ **NEVER** apply opacity changes or transparency
- ❌ **NEVER** rotate or skew the logo
- ❌ **NEVER** apply gradients or texture overlays
- **Rule:** Logo must remain clear, flat, and unmodified

### 5.6 Background Restrictions
- ❌ **NEVER** place color logo on dark or saturated backgrounds
- ❌ **NEVER** place white logo on white or light backgrounds
- ❌ **NEVER** use gradient backgrounds that reduce contrast
- **Rule:** Ensure sufficient contrast (WCAG AA minimum)

## 6. LIGNE D'ÉTAT (DETAILED SPECIFICATIONS)

### 6.1 Color Composition

The Ligne d'État is a **FIXED tricolor bar** representing the RDC flag:

| Section | Color | Position | Percentage | Hex Code |
|---------|-------|----------|-----------|----------|
| **Top (Bleu)** | Blue | Superior | 33.33% | #0095C9 |
| **Middle (Jaune)** | Yellow | Central | 33.33% | #FFF24B |
| **Bottom (Rouge)** | Red | Inferior | 33.33% | #DB3832 |

**Rule:** Each color occupies exactly one-third. No color may exceed 33.33%.

### 6.2 Orientations

#### Vertical (Primary)
- **Usage:** Main logo separator, vertical layouts
- **Width:** 4px (digital) | 1mm (print)
- **Height:** Variable, scales with logo
- **Asset:** `ligne_etat.png` | `ligne_etat_white.png`

#### Horizontal (Secondary)
- **Usage:** Decorative, header accents, dividers
- **Height:** 4px (digital) | 1mm (print)
- **Width:** Variable, adapts to context
- **Asset:** `ligne-etat-h` (CSS class-based) or image version

### 6.3 Asset Implementation

**Always use image assets, never CSS-based implementations:**

```html
<!-- CORRECT: Image-based -->
<img src="ligne_etat.png" alt="Ligne d'État"
     style="width:4px;height:50px;object-fit:contain;flex-shrink:0;">

<!-- Light background: use colored -->
<img src="../logo_assets/ligne_etat.png" alt="Ligne d'État">

<!-- Dark background: use white -->
<img src="../logo_assets/ligne_etat_white.png" alt="Ligne d'État">

<!-- WRONG: CSS-based (deprecated) -->
<span class="ligne-etat"></span>  <!-- ❌ Avoid -->
```

### 6.4 Styling Rules

When implementing Ligne d'État images:

```css
/* CORRECT */
img[alt="Ligne d'État"] {
  object-fit: contain;      /* Prevent distortion */
  flex-shrink: 0;           /* Maintain proportions */
  width: 4px;               /* X unit */
  height: auto;             /* Or specific value */
}

/* Responsive sizing */
@media (max-width: 768px) {
  img[alt="Ligne d'État"] {
    width: 2-3px;           /* Scale down width proportionally */
    /* Height remains relative to context */
  }
}
```

### 6.5 Standalone Ligne d'État Usage

When used decoratively (not as logo separator):

- ✅ **Allowed:** Header accents, vertical dividers, sidebar elements
- ✅ **Allowed:** Hover states, transitions, highlights
- ✅ **Allowed:** Multiple small instances (decorative pattern)
- ✅ **Allowed:** Background accents with reduced opacity (max 50%)
- ⚠️ **Caution:** Ensure context doesn't suggest official logo use
- ❌ **Forbidden:** Distort, recolor, or invert as a logo substitute

## 7. LAYOUT & DISPOSITION

### 7.1 Logo Placement Priorities

1. **Header/Navigation:** Logo appears in top-left or center
2. **Alignment:** Horizontal center or left-aligned (never right, except in RTL languages)
3. **Vertical Position:** Top 16px (4X) from content boundary
4. **Horizontal Position:** 16px (4X) from left edge

### 7.2 Sidebar Logo Requirements

All design system pages **MUST** have consistent sidebar navigation with:

```
Accueil
├── Home

Tokens
├── Couleurs
├── Typographie
├── Espacements
├── Ombres

Composants
├── Boutons
├── Formulaires
├── Cartes
├── Navigation
├── Tableaux
├── Badges & Alertes
├── En-tête Officiel

Marque
├── Logo
├── Ligne d'État

Modèles
├── Page d'Accueil
├── Page Intérieure
├── Page Service
```

**Rule:** "Marque" section comes BEFORE "Modèles" section in sidebar order.

### 7.3 Component Header Layout

Official headers must follow this structure:

1. **Logo Area (Left):** Armoiries + Ligne d'État + Intitulé
2. **Spacing:** 4X (16px) between elements
3. **Alignment:** Horizontal center within its container
4. **Height:** Minimum 50px for web headers
5. **Background:** White (light) or institutional color (dark)

### 7.4 Documentation Page Layout

All design system documentation pages must maintain:

- Consistent sidebar on left (collapsed/expanded)
- Main content area with sections
- Page hero with title and description
- Logical section hierarchy (h2, h3, etc.)
- Visual examples with code references
- Badge indicators (Recommended, Authorized, Restricted, Usage, Forbidden)

## 8. TECHNICAL IMPLEMENTATION

### 8.1 Asset Files

All logo assets must be located in: `/logo_assets/`

| File | Purpose | Format | Usage |
|------|---------|--------|-------|
| `armoiries.png` | Coat of Arms (colored) | PNG | Light backgrounds |
| `armoiries_white.png` | Coat of Arms (white mono) | PNG | Dark backgrounds |
| `ligne_etat.png` | Tricolor bar (colored) | PNG | Light backgrounds |
| `ligne_etat_white.png` | Tricolor bar (white mono) | PNG | Dark backgrounds |

### 8.2 CSS Variables (Design Tokens)

Required CSS variables for branding:

```css
/* Color palette */
--rdc-bleu: #0095C9;
--rdc-jaune: #FFF24B;
--rdc-rouge: #DB3832;
--rdc-bleu-inst: #1A3A6B;  /* Institutional blue */
--rdc-bleu-fonce: darker variant

/* Spacing (X unit) */
--space-x: 4px;
--space-4x: 16px;  /* Security zone */

/* Typography */
--font-principale: "Inter", sans-serif;
--font-complement: "Outfit", sans-serif;
```

### 8.3 Responsive Breakpoints

Logo sizing at each breakpoint:

```css
/* Mobile < 768px */
@media (max-width: 767px) {
  .logo { height: 40px; }
}

/* Tablet 768px - 1023px */
@media (min-width: 768px) and (max-width: 1023px) {
  .logo { height: 50px; }
}

/* Desktop >= 1024px */
@media (min-width: 1024px) {
  .logo { height: 110px; }  /* or context-dependent */
}
```

### 8.4 Accessibility (WCAG 2.1 AA)

- **Alt Text:** All logo images must have descriptive alt text
  - `alt="Armoiries RDC"` for coat of arms
  - `alt="Ligne d'État"` for tricolor bar
  - `alt="Logo officiel - Ministère du Numérique"` for complete logo
- **Color Contrast:** Minimum 4.5:1 for text on logo backgrounds
- **Keyboard Navigation:** Logo links must be keyboard accessible
- **Language:** Alt text and descriptions in page language (FR/EN)

### 8.5 Bilingual Considerations

- **French Names:** Use current French ministry/department names
- **English Translation:** Provide English equivalents in documentation
- **Text Direction:** Left-to-right for both FR and EN (French is not RTL)
- **Font Sizing:** Account for text length differences (FR typically longer)
- **Line Breaks:** Test both languages for overflow/wrapping

## 9. COMMON ERRORS & CORRECTIONS

### Error 1: CSS-Based Ligne d'État

❌ **WRONG:**
```html
<span class="ligne-etat"></span>
```

✅ **CORRECT:**
```html
<img src="../logo_assets/ligne_etat.png" alt="Ligne d'État"
     style="width:4px;height:50px;object-fit:contain;flex-shrink:0;">
```

### Error 2: Logo Proportions Too Large

❌ **WRONG:** Using 50px logo height in card examples

✅ **CORRECT:** Scale by context
- Card examples: 35px
- Section headers: 40px
- Page headers: 50px
- Social media: 110px

### Error 3: Old Government Text

❌ **WRONG:**
```html
<span>République Démocratique du Congo</span>
```

✅ **CORRECT:**
```html
<span>Ministère du Numérique</span>  <!-- Or current ministry name -->
```

### Error 4: Wrong Asset Versions

❌ **WRONG:**
```html
<!-- Light background, using white assets -->
<img src="armoiries_white.png"> on white background
```

✅ **CORRECT:**
```html
<!-- Light background: colored assets -->
<img src="armoiries.png">
<img src="ligne_etat.png">

<!-- Dark background: white assets -->
<img src="armoiries_white.png">
<img src="ligne_etat_white.png">
```

### Error 5: Inverted Colors

❌ **WRONG:**
```css
/* Inverted color order */
gradient: 180deg, rouge, jaune, bleu
```

✅ **CORRECT:**
```css
/* Top to bottom: bleu → jaune → rouge */
gradient: 180deg, #0095C9, #FFF24B, #DB3832
```

## 10. COMPLIANCE CHECKLIST

Use this checklist when creating or modifying design system pages:

- [ ] Logo consists of all three elements (Armoiries + Ligne + Intitulé)
- [ ] Ligne d'État uses image assets, not CSS classes
- [ ] Correct asset versions used (colored for light, white for dark backgrounds)
- [ ] Logo proportions follow sizing guidelines (53px minimum height)
- [ ] Security zone (4X = 16px) respected around logo
- [ ] No "République Démocratique du Congo" text (use ministry names)
- [ ] Sidebar navigation in correct order (Marque before Modèles)
- [ ] All images have proper alt text for accessibility
- [ ] Color contrast minimum 4.5:1 for text
- [ ] Tested in both French and English versions
- [ ] No color inversions or distortions applied
- [ ] Page follows standard design system layout (hero, sections, cards)
- [ ] Responsive sizing tested at all breakpoints
