Boutons

Composants interactifs pour les actions principales, secondaires et contextuelles.

Bouton Principal (Primary) — Tailles

Le bouton principal utilise le dégradé bleu institutionnel. Disponible en 3 tailles : small, default et large.

<button class="btn btn-primary btn-sm">Petit bouton</button>
<button class="btn btn-primary">Bouton par défaut</button>
<button class="btn btn-primary btn-lg">Grand bouton</button>

Bouton Secondaire

Pour les actions secondaires. Fond léger avec bordure bleue.

<button class="btn btn-secondary">Secondaire</button>

Bouton Outline

Bouton avec bordure uniquement, fond transparent.

<button class="btn btn-outline">Outline</button>

Bouton Ghost

Bouton discret sans bordure ni fond, idéal pour les actions tertiaires.

<button class="btn btn-ghost">Ghost</button>

Bouton Danger

Pour les actions destructives ou irréversibles. Utilise le rouge du drapeau.

<button class="btn btn-danger">Supprimer</button>

Boutons Icône

Boutons carrés contenant uniquement une icône. Combinables avec toutes les variantes.

<button class="btn btn-primary btn-icon">
  <i data-lucide="plus"></i>
</button>
<button class="btn btn-secondary btn-icon">
  <i data-lucide="search"></i>
</button>
<button class="btn btn-outline btn-icon">
  <i data-lucide="settings"></i>
</button>
<button class="btn btn-ghost btn-icon">
  <i data-lucide="x"></i>
</button>
<button class="btn btn-danger btn-icon">
  <i data-lucide="trash-2"></i>
</button>

État Désactivé

L'attribut disabled réduit l'opacité et désactive les interactions.

<button class="btn btn-primary" disabled>Primary désactivé</button>
<button class="btn btn-secondary" disabled>Secondary désactivé</button>

État Chargement

La classe btn-loading masque le texte et affiche un spinner centré.

<button class="btn btn-primary btn-loading">Chargement</button>

Bouton avec Icône et Texte

Combinez texte et icône dans un bouton standard grâce au gap intégré.

<button class="btn btn-primary">
  <i data-lucide="download"></i> Télécharger
</button>

Groupe de Boutons

Alignez plusieurs boutons dans un groupe horizontal pour les actions liées.

<!-- Groupe connecté -->
<div style="display:flex;gap:0;">
  <button class="btn btn-outline" style="border-radius:12px 0 0 12px;border-right:none;">Jour</button>
  <button class="btn btn-primary" style="border-radius:0;">Semaine</button>
  <button class="btn btn-outline" style="border-radius:0 12px 12px 0;border-left:none;">Mois</button>
</div>

<!-- Groupe espacé -->
<div style="display:flex;gap:12px;">
  <button class="btn btn-primary">Enregistrer</button>
  <button class="btn btn-secondary">Annuler</button>
  <button class="btn btn-ghost">Réinitialiser</button>
</div>
