Formulaires

Champs de saisie, sélecteurs et contrôles pour la collecte de données.

Champ Texte — États

Le champ texte supporte plusieurs états visuels : par défaut, focus, erreur, succès et désactivé.

Ce champ montre l'état focus avec la bordure bleue.

Ce champ contient une erreur. Veuillez corriger.

Valide !

<input type="text" class="input" placeholder="Par défaut">
<input type="text" class="input input--error" value="texte invalide">
<input type="text" class="input input--success" value="texte valide">
<input type="text" class="input" disabled value="Non modifiable">

Zone de Texte (Textarea)

Pour les saisies multi-lignes comme les commentaires ou descriptions.

Maximum 500 caractères.

<label class="form-label">Message</label>
<textarea class="input textarea" placeholder="Écrivez votre message ici..."></textarea>
<p class="form-hint">Maximum 500 caractères.</p>

Menu Déroulant (Select)

Sélecteur personnalisé avec flèche intégrée.

<label class="form-label">Province</label>
<select class="input select">
  <option value="">Sélectionnez une province</option>
  <option value="kinshasa">Kinshasa</option>
  <option value="katanga">Haut-Katanga</option>
</select>

Cases à Cocher (Checkbox)

Cases à cocher avec styles personnalisés. L'état coché utilise la couleur primaire.

<label>
  <input type="checkbox" class="checkbox"> Non coché
</label>
<label>
  <input type="checkbox" class="checkbox" checked> Coché
</label>

Boutons Radio

Pour la sélection unique dans un groupe d'options.

<label>
  <input type="radio" name="genre" class="radio" checked> Masculin
</label>
<label>
  <input type="radio" name="genre" class="radio"> Féminin
</label>

Interrupteur (Toggle)

Commutateur booléen pour activer ou désactiver une option.

<label>
  <input type="checkbox" class="toggle"> Notifications push
</label>
<label>
  <input type="checkbox" class="toggle" checked> Mode sombre
</label>

Disposition : Label + Input + Hint + Erreur

Structure recommandée pour un champ de formulaire complet.

Utilisez votre adresse professionnelle @gouv.cd.

Le mot de passe doit contenir au moins 8 caractères.

<div>
  <label class="form-label">Adresse email <span style="color:var(--color-danger);">*</span></label>
  <input type="email" class="input" placeholder="exemple@gouv.cd">
  <p class="form-hint">Utilisez votre adresse professionnelle.</p>
</div>
<div>
  <label class="form-label">Mot de passe</label>
  <input type="password" class="input input--error">
  <p class="form-error-msg">Le mot de passe doit contenir au moins 8 caractères.</p>
</div>
