Ligne d'État

La barre tricolore qui symbolise l'unité nationale et l'autorité de l'État congolais.

Définition

La Ligne d'État est la barre tricolore verticale reprenant les couleurs du drapeau national. Elle symbolise la rigueur, la détermination et l'unité nationale.

Présente dans le logo officiel comme séparateur entre le Bloc-armoirie et l'Intitulé, elle s'utilise également de manière autonome comme élément décoratif et identitaire sur tous les supports de communication gouvernementaux.

Composition des couleurs

Bleu #0095C9 · 33.33%
Haut
Jaune #FFF24B · 33.33%
Milieu
Rouge #DB3832 · 33.33%
Bas
Les trois couleurs occupent exactement un tiers chacune (33.33%). Aucune couleur ne doit être plus grande qu'une autre. L'ordre de lecture est toujours du haut vers le bas : Bleu, Jaune, Rouge.

Documentation

Propriétés CSS obligatoires

  • object-fit: contain — Prévient la déformation
  • flex-shrink: 0 — Maintient les proportions (en flexbox)
  • alt="Ligne d'État" — Accessibilité WCAG
  • Width: 4px (vertical) | Height: 4px (horizontal)
  • Min-height: 53px minimum pour usage autonome
  • Pas de: ombres, rotations, opacité, filtres

Checklist de conformité

Avant d'utiliser la Ligne d'État dans votre implémentation, assurez-vous que tous ces critères sont respectés.

Couleurs

  • ☐ Ordre: Bleu (haut) → Jaune (milieu) → Rouge (bas)
  • ☐ Chaque couleur occupe 33.33% exactement
  • ☐ #0095C9 (Bleu) | #FFF24B (Jaune) | #DB3832 (Rouge)
  • ☐ Jamais d'inversion ou de réordonner les couleurs

Orientations

  • ☐ Verticale: Logo, séparateur vertical (standard)
  • ☐ Horizontale: Header border, séparateur section
  • ☐ Jamais de rotation à 45°, 90°, etc.
  • ☐ Toujours perpendiculaire (0° ou 90°)

Proportions

  • ☐ Width: 4px (digital) | 1mm (print)
  • ☐ Min-height: 53px (autonome)
  • ☐ Jamais d'aspect ratio déformé
  • ☐ Height = Armoirie height (dans logo)

Usages autorisés

  • ☐ Séparateur dans le logo officiel
  • ☐ Bordure de header/footer
  • ☐ Séparateur de sections de contenu
  • ☐ Élément décoratif dans sidebars

Assets

  • ☐ Fond clair: ligne_etat.png
  • ☐ Fond sombre: ligne_etat_white.png
  • ☐ Jamais mélanger les assets
  • ☐ Assets en logo_assets/

Accessibilité

  • ☐ alt="Ligne d'État" sur les images
  • ☐ Contraste ≥ 4.5:1 (WCAG AA)
  • ☐ Pas d'opacité réduisant le contraste
  • ☐ object-fit:contain
