Identité

Ligne d'État

La barre tricolore qui symbolise l'unité nationale et l'autorité de l'État congolais.

Définition

La Ligne d'État est la barre tricolore verticale reprenant les couleurs du drapeau de la République Démocratique du Congo. Elle symbolise la rigueur, la détermination et l'unité nationale.

Présente dans le logo officiel comme séparateur entre le Bloc-armoirie et l'Intitulé, elle s'utilise également de manière autonome comme élément décoratif et identitaire sur tous les supports de communication gouvernementaux.

Composition des couleurs

Bleu #0095C9 · 33.33%
Haut
Jaune #FFF24B · 33.33%
Milieu
Rouge #DB3832 · 33.33%
Bas
Les trois couleurs occupent exactement un tiers chacune (33.33%). Aucune couleur ne doit être plus grande qu'une autre. L'ordre de lecture est toujours du haut vers le bas : Bleu, Jaune, Rouge.

Orientations

Standard

Verticale

Orientation par défaut. Utilisée dans le logo, les sidebars et comme séparateur vertical.

Variante

Usages

Dans le logo officiel

Séparation entre le Bloc-armoirie et l'Intitulé officiel.

Bordure inférieure du header

Version horizontale en bas de l'en-tête officiel.

Séparateur de sections

Version horizontale pour diviser les grandes sections de contenu.

Élément décoratif vertical

Dans les sidebars et panneaux latéraux comme accent visuel.

Code CSS

marque.css CSS 
/* Ligne d'État verticale */
.ligne-etat {
  display: block;
  width: 4px;
  min-height: 53px;
  background: linear-gradient(
    180deg,
    #0095C9  0%,
    #0095C9  33.33%,
    #FFF24B 33.33%,
    #FFF24B 66.66%,
    #DB3832 66.66%,
    #DB3832 100%
  );
}

/* Ligne d'État horizontale */
.ligne-etat-h {
  display: block;
  height: 4px;
  width: 100%;
  background: linear-gradient(
    90deg,
    #0095C9  0%,
    #0095C9  33.33%,
    #FFF24B 33.33%,
    #FFF24B 66.66%,
    #DB3832 66.66%,
    #DB3832 100%
  );
}
