Logo & Identité

Architecture, règles de composition et usages du logo officiel.

Architecture du Logo

Le logo officiel se compose de trois éléments indissociables. Aucun ne peut être utilisé seul ou modifié indépendamment.

Ligne d'État Séparateur tricolore

Ministère du Numérique

Interdits

Ces usages sont strictement interdits et constituent une violation de la charte graphique.

Ne pas déformer les proportions

Le logo ne doit jamais être étiré, compressé ou déformé.

Ne pas changer l'ordre des couleurs

L'ordre Bleu → Jaune → Rouge doit toujours être respecté.

Implémentation technique

Toutes les implémentations doivent utiliser les assets image dans logo_assets/, jamais des versions CSS pour le logo officiel.

Sur fond blanc/clair

  <img src="armoiries.png"
       alt="Armoiries RDC"
       style="width:50px;
               height:50px;
               object-fit:contain;
               flex-shrink:0;">

Fonds foncés

Propriétés CSS essentielles

  • object-fit: contain — Prévient la déformation
  • flex-shrink: 0 — Maintient les proportions
  • alt="Description" — Accessibilité WCAG
  • Jamais d'effets : ombres, rotations, opacité

Checklist de conformité

Avant de publier une page ou un composant utilisant le logo officiel, assurez-vous que tous les critères suivants sont respectés.

Architecture

  • ☐ Logo contient 3 éléments : Armoiries + Ligne + Intitulé
  • ☐ Aucun élément n'est utilisé seul (sauf décoratif)
  • ☐ Intitulé officiel est un vrai ministère/département
  • ☐ Pas de "République Démocratique du Congo" texte

Composition

  • ☐ Hauteur Ligne d'État = Hauteur Armoiries
  • ☐ Espacement 4X (16px) respecté entre éléments
  • ☐ Zone de sécurité 4X (16px) autour du logo
  • ☐ Logo height ≥ 50px (web) ou 53px (minimum)

Couleurs & Assets

  • ☐ Fonds clairs: armoiries.png + ligne_etat.png
  • ☐ Fonds foncés: armoiries_white.png + ligne_etat_white.png
  • ☐ Ligne d'État: Bleu (33%) → Jaune (33%) → Rouge (33%)
  • ☐ Jamais d'inversion ou réordonner les couleurs

Interdits

  • ☐ ✘ Jamais déformer (étirer, comprimer)
  • ☐ ✘ Jamais appliquer ombres/glows/opacité
  • ☐ ✘ Jamais changer couleurs ou police
  • ☐ ✘ Jamais placer en couleur sur fond foncé

Accessibilité

  • ☐ Tous les <img> ont un alt descriptif
  • ☐ Contraste couleur/texte ≥ 4.5:1 (WCAG AA)
  • ☐ Logo est un lien? → Keyboard accessible
  • ☐ object-fit:contain + flex-shrink:0

Responsive

  • ☐ Mobile (<768px): 35-40px logo height
  • ☐ Tablet (768-1024px): 50px logo height
  • ☐ Desktop (>1024px): 50-110px (contexte)
  • ☐ Proportion maintenue à tous les breakpoints
